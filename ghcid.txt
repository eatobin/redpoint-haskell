Correct way to run ghcid

[eric@eric-minisforum ~]$ cd haskell-projects/redpoint-haskell

Build/install:
[eric@eric-minisforum day01](master)$ stack build --copy-compiler-tool ghcid
[eric@eric-minisforum day01](master)$ stack exec -- which ghcid

Run:
[eric@eric-minisforum day01](master)$ stack exec -- ghcid -c "stack ghci redpoint-haskell"
[eric@eric-minisforum day01](master)$ stack exec -- ghcid -c "stack ghci test/Spec.hs"
